# pizza-mario
pizza mario is a frontend web application design to order pizza online

### `features`
The application has the following features
<ul>
  <li>Login</li>
  <li>Signup</li>
  <li>Dashboard</li>
  <li>menu - (multiple links)</li>
  <li>cart - (local storage)</li>
  <li>Bill calculation</li>
</ul>
